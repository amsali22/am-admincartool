# am-admincartool
admin car tool, will help you manage players cars.


# Preview
[Link](https://streamable.com/5ru7xl)

# Installation
* Drag and drop `am-admincartool`
* In your `server.cfg` add `ensure am-admincartool`

# How To use
* `/playercars` or `/playercars CID`  (Must be Admin)

# Dependencies
ox_lib
oxmysql
